# IPL DASHBOARD PROJECT

## Project Overview

This project is an interactive IPL Dashboard built using Python, Streamlit, Pandas, and Plotly. The dashboard analyzes IPL match data and provides insights through interactive visualizations and filters.

---

## Features

* Dataset Overview
* Data Cleaning
* Key Performance Indicators (KPIs)
* Interactive Filters
* Winning Team Analysis
* Toss Decision Analysis
* Data Visualization using Plotly

---

## Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Streamlit

---

## Dataset

* IPL Matches Dataset (CSV format)

---

## Visualizations Included

* Winning Teams Analysis
* Toss Decision Analysis
* Season-wise Match Analysis
* Venue Analysis
* Player of Match Analysis

---

## How to Run the Project

* Step 1: Install required libraries using:
  `pip install streamlit pandas plotly`

* Step 2: Run the Streamlit app using:
  `streamlit run app.py`
